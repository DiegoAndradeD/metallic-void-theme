**Metallic Void** is a dark, minimalist VS Code theme crafted around a specific personal taste for **matte** and **metallic** colors.

It avoids neon or overly saturated tones in favor of a "dry," industrial look that reduces eye strain during long coding sessions.

## Installation (Manual)

Since this theme is not yet on the Marketplace, you can install it manually:

1. Clone this repository or download the folder.
2. Copy the `metallic-void` folder to your VS Code extensions directory:
   - **Linux/Mac:** `~/.vscode/extensions/`
   - **Windows:** `%USERPROFILE%\.vscode\extensions\`
3. Restart VS Code.
4. Open the Command Palette (`Ctrl+Shift+P`), type **"Preferences: Color Theme"**, and select **Metallic Void**.
